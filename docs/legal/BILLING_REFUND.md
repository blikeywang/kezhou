# 计费与退款条款(草稿 · Draft)

> ⚠️ **模板草稿,非法律意见。** 数字商品退款、自动续费披露、加密支付、跨境税务在各辖区规则不同(如欧盟消费者撤回权、中国《消费者权益保护法》、美国各州自动续费法 ARL/加州 auto-renewal 等),**必须律师审**。占位项以 `[方括号]` 标注。

价格、时长与含义以定价页为准;我们保留调整权,调整不影响已购周期。

## 2. 各支付方式的激活与续费
- **闲鱼兑换码**:在闲鱼付款后凭发放的**卡密**在站内兑换激活;卡密一次性、绑定档位与有效期,不记名、请妥善保管。**为一次性购买,不自动续费**;到期需重新购买。
- **Stripe**:采用**自动续费订阅**;将在每个计费周期开始时自动扣款,直至你在 [账户/Customer Portal] 取消。取消在**当前周期结束时生效**,当期不按比例退款(除非法律另有强制规定)。
- **加密支付**:为**一次性购买**,按提交交易的**链上确认**激活;不自动续费,到期需重新购买。

## 3. 退款政策
- 本服务为**数字信息服务**,一经开通即时可用。除法律强制赋予的撤回/退款权外,原则上**不予退款**。
- **法定撤回权**:若适用地区法律赋予冷静期/撤回权,你可在规定期限内依法申请;但你**明确同意**在购买后立即开通并使用即时数字内容,可能导致撤回权丧失(以当地法律为准)。
- **误扣/重复扣款/技术故障**导致的错误收费,经核实后退款。
- **闲鱼**渠道的争议与退款,依闲鱼平台规则处理。
- **加密支付**因链上不可逆,原则上**不退**;确因我方原因未开通的,按**原币种**退回(汇率波动风险由此产生的差额不承担)。

## 4. 自动续费披露(Stripe)
订阅将**自动续费并扣款**,续费价格、周期与取消方式在结账页与账户页明确展示;我们会在续费前依法/依渠道要求发送提醒。你可随时取消,取消后不再产生新的扣款。

## 5. 税费与发票
价格是否含税以结账页为准;跨境交易可能产生增值税/销售税等,由相应规则确定。[发票/收据获取方式]。
